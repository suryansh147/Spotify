A Spotify-style music player built using HTML, CSS, and JavaScript.  
It lets users browse albums and play local MP3 files with basic controls.

This is a frontend clone for learning purposes only.

## Features

- Album browsing  
- Playlist view  
- Play / Pause / Next / Previous  
- Seek bar  
- Volume control  
- Responsive sidebar
